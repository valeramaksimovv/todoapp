# TodoApp for TEB Education

### Version 0.1.0

## To run this application, you need to do the following

1. Install Docker
2. Download the code to your computer or server
3. Open the terminal in the folder with the code
4. Rename the .env.example file to .env
5. Run the command “docker compose up --build”
6. Go to 0.0.0.0:8000
